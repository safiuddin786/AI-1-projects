To run the program you can use any IDE or the terminal or CMD.
Python version 3.11.0 is recommended but can run on python version >3.7.0

Command to type in terminal
   python3 nim.py <number of red balls> <number of blue balls> <first-turn> <depth>
Note: instead of python3 you can type python in above command
first-turn: It is an optional argument. It can have two values 'human' or 'computer'
depth: depth is also an optional argument. It limits the search to that depth.
